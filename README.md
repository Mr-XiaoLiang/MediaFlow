# Media Flow

- 本地视频、图片按文件夹添加
- 隐私模式的视频与图片独立管理
- 类短视频的播放模式（播放页面上下滑动即可切换）
- 完全的手势处理，播放页面允许通过手势快进快退，并且支持变速快进
- 快速的预加载能力，接近无感的视频切换
- 瀑布流的视频列表页面
- 播放页面支持侧边栏快速定位
- 播放页面支持宫格的视频切换
- 完全的横竖屏兼容，允许播放过程中切换横竖屏
- 多文件夹的归档功能，提供额外的归档管理页面
- 覆盖全面的UI显示状态设置
- 视频字幕的支持
- 流式图片展示，无缝浏览图片，方便漫画查看
